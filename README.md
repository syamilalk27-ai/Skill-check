# CURE Skill Check Simulator

Skill check simulator berbasis mekanik dari CURE Violence District.

## Deploy ke GitHub Pages

1. Upload semua file ini ke repository GitHub
2. Masuk ke **Settings → Pages**
3. Source: pilih **main branch**
4. Website live di: `https://username.github.io/nama-repo`

## Fitur
- Skill check wheel akurat (zona GREAT, SUCCESS, FAIL)
- Auto Perfect mode
- Speed multiplier
- Score & streak tracker
